== Description ==

**CulturalTranslate** brings professional, AI-powered translation with cultural context awareness directly to your WordPress site.

### Key Features

* **AI-Powered Translation** - Intelligent translation with GPT-5 powered cultural context analysis
* **7-Layer Context Analysis** - Domain, formality, audience, cultural sensitivity, idioms, references, and intent
* **Brand Voice** - Maintain consistent brand voice across all translations
* **Glossary Support** - Custom terminology management
* **Post/Page Translation** - Translate WordPress content with one click
* **Shortcodes** - `[culturaltranslate]` and `[ct_verify]` for inline translation and certificate verification
* **Gutenberg Block** - Native block editor support
* **Widgets** - Translation widget and language switcher
* **Certificate Verification** - Verify translation certificates
* **Real-time Translation** - Instant translation in the WordPress admin

### Use Cases

* **Multilingual Websites** - Translate your entire website
* **E-commerce** - Translate product descriptions
* **Blogs** - Translate blog posts
* **Documentation** - Translate documentation
* **Legal Documents** - Professional legal translation

### Requirements

* WordPress 5.8 or higher
* PHP 7.4 or higher
* CulturalTranslate API Key (get it at https://culturaltranslate.com)

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/culturaltranslate/`
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to CulturalTranslate > Settings
4. Enter your API Key
5. Configure default languages
6. Start translating!

== Frequently Asked Questions ==

= Do I need an API key? =

Yes, you need a CulturalTranslate API key. Sign up at https://culturaltranslate.com to get your free API key.

= How many languages are supported? =

CulturalTranslate supports 100+ languages including English, Arabic, French, Spanish, German, Chinese, Japanese, and more.

= Can I translate WooCommerce products? =

Yes! The plugin works with WooCommerce and other custom post types.

= Is translation cached? =

Yes, translations are cached for 24 hours by default to improve performance and reduce API calls.

= Can I customize the translation? =

Yes, you can edit translations manually after they're generated.
