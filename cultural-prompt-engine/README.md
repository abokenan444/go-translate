# Cultural Prompt Engine for CulturalTranslate

This package adds a full prompt-orchestration layer to your Laravel project.

It provides:

- Cultural Profiles (per country / culture)
- Emotional Mapping (tones)
- Industry Library (per sector)
- Task Templates (per use-case)
- Unified Prompt Generator (compiles everything into a single OpenAI-ready prompt)
- Post-processing Filters (for safety and brand rules)

You can install it by running:

```bash
bash install.sh
```

from the **root of your Laravel project**.
